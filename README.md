# CocomoMdS

Programma in python che determina il linguaggio in cui è scritto un file di codice e controlla se corrisponde all'estensione.

## Esecuzione

Aprire un terminale e digitare il seguente comando:
- Windows:
  ```
  python main.py
  ```

- Linux:
  ```
  python3 main.py
  ```
